# AssistantNet

AI-powered autonomous office assistant portal. Manages emails, calendar, tasks, documents, and analytics with intelligent workflow automation — ready to replace a human office manager from day one.

## Features

- **🧠 AI Command Center** — Streaming LLM chat with business-aware context
- **📧 Smart Inbox** — AI email triage, priority sorting, auto-draft replies
- **📅 Calendar** — Week view with conflict detection and smart scheduling
- **✅ Task Engine** — Kanban board with drag-and-drop and AI task breakdown
- **📄 Document Hub** — Search, AI summarization, and template generation
- **📊 Dashboard** — Real-time KPIs, analytics charts, and AI activity feed
- **🤖 Autonomous Mode** — AI proactively handles queued work items

## Quick Start

```bash
npm install
npm run dev
```

## LLM Integration

AssistantNet works out of the box with intelligent built-in responses. To enable live Gemini AI:

1. Get an API key from [Google AI Studio](https://aistudio.google.com/)
2. The app uses `gemini-2.0-flash` for fast, intelligent responses

## Tech Stack

- **Vite** — Build tool
- **Vanilla JS** — Zero framework overhead
- **Chart.js** — Analytics visualizations
- **Marked** — Markdown rendering
- **Google GenAI** — LLM integration (optional)
